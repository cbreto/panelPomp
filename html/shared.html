shared {panelPomp}R Documentation

Extract shared parameters from a panelPomp object

Description

This function is used to extract the shared parameters from a panelPomp object.

Usage

shared(object, ...)

Arguments

object

an object that contains shared parameters.

...

additional arguments.

Value

vector containing the shared parameters

Author(s)

Jesse Wheeler

See Also

panelPomp_methods

Examples

prw <- panelRandomWalk()
# extract parameters in list form
shared(prw)
